# tf-root-aws-cookiecutter

[Cookiecutter](https://www.cookiecutter.io/) template to generate a basic root
Terraform module for AWS.

## Pre-requisites

```sh
pip install pipx
```

## Run

```sh
pipx run cookiecutter gh:andreswebs/tf-root-aws-cookiecutter
```
